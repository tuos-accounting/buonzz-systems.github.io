Get Started

The software is incredibly flexible in terms of deployment capability, so that you can run it in your local computer, local network or online (self-hosted or Cloud). If you are just trying to see how the software looks like, you can take a look at our online demo.

If you don't have a dedicated IT staff that can install it for you, you can avail of our one-time paid Installation Service and we will set it up for you.

Or You can choose any deployment option below:

Choose Deployment Option

Desktop Client (Standalone application - RECOMMENDED)

  • Install in my Desktop computer (PC/MacOS) locally - great for small business who doesn't conduct online transaction and just wanted a simple desktop app.

Local Network (Linux Server)

  • Install in my local Linux Server - On-Premise installations, wherein you plan to serve it in multiple users in your Intranet.

Cloud

The following option is great for business who wants to host it online and the users needs to access it via Internet.

  • Deploy in Heroku - if you don't have your own server and don't know how to setup one.
  • Deploy in AWS - if you know how to setup EC2 instances and RDS.
  • Deploy in Google Cloud - if you prefer to deploy using Compute Engine and Cloud SQL
  • Deploy in Bare Metal - your own VPS server, DigitalOcean, Azure VM, Linode etc.

Docker

This deployment is ideal for corporate and huge enterprise deployment wherein you need to scale it out and you expect to handle enormous amount of data.

  • Install via Docker
  • Install on a Kubernetes Cluster
  • Install on Docker Swarm
